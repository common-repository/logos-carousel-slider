Logos Carousel slider is very helpful to display clients, partners and affiliates logos. You can create shortcode and use it. thats simple. It is mobile friednly.

== Description ==
	This plugin is used to create sliding logos. You can add name, featured image and category. just use the [Logos_Carousel] short code in required section and it will work fine.  One of the amazing features of this plugin is that it has touch and swipe support. That means your users from any devices can view the logo carousel/slider nicely.

Features

    Fully Responsive
	1 layouts 
    Slider control
	Very lightweight.
	Unlimited logos display.
    Widget Supported 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/Logos Carousel Slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You see the admin menu option name "Logo Slider".
